局部变量和成员变量的区别

语法形式

  • 从语法形式来看，成员变量是定义在类中，局部变量是在代码块或方法中定义的变量和或者参数，成员变量可以被权限修饰符和 static 修饰，而局部变量不可以被权限修饰符 和 static 修饰，局部变量和成员变量都可以被 final 关键字修饰

存储方式

  • 从存储方式来看，如果成员变量被 static 修饰，那么该成员变量是属于类，如果成员变量没有被 static 修饰，那么该成员变量是属于实例，而对象是存储在堆内存，局部变量存储在栈内存

注意：如果局部变量没有被 static 修饰，那么局部变量是存储在虚拟栈的局部变量表当中，如果被 static 修饰，那么是存储在

生存时间

  • 从变量在内存的生存时间来看，成员变量是对象的一部分，它随着对象的创建而存在，随着对象的销毁而消失，而局部变量是与方法的调用有关，局部变量随着方法的调用而创建，当方法调用结束，局部变量也就被销毁了

默认值

  • 从变量是否有默认值来看，成员变量如果没有赋值，会有相应数据类型的默认值，如实例对象如果没有赋值，默认为 null，但是局部变量如果没有赋值是不会有默认值的，在编译时会报错
